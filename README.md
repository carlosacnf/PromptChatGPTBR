# Uma coleção de prompts ChatGPT, GPT-3.5, GPT-4

Este projeto consiste em prompts para os modelos ChatGPT e GPT-3.5 que são projetados para ajudar com tarefas de escrita, análise e compreensão. Há muitos prompts diferentes disponíveis para você usar, e você pode usá-los para gerar conteúdo para seus projetos, depurar seu código, encontrar soluções para problemas ou simplesmente aprender mais sobre o que esses modelos podem fazer. Com esses prompts, você guiará os modelos para resolver qualquer tarefa relacionada ao idioma usando os verbos instrucionais corretos.

[InstruirGPT] (https://arxiv.org/abs/2203.02155) (versões mais recentes referidas como GPT-3.5) são modelos de linguagem de série que foram treinados usando feedback humano para entender melhor e alinhar com a intenção de um usuário, produzindo saídas mais precisas e apropriadas.

[ChatGPT] (https://openai.com/blog/chatgpt/) também usa o método instruGPT, mas em forma de diálogo para entender as instruções do usuário e gerar saídas com base na instrução do usuário.

[GPT4] (https://openai.com/research/gpt-4) Mais poderoso do que qualquer modelo GPT-3.5, ele pode lidar com instruções mais complexas e pode segui-las e aplicá-las de forma mais eficaz.

## Por que usar:

Este é um método fácil e direto para orientar o modelo a fazer quase qualquer coisa. Ele usa uma estrutura simples para fornecer instruções e pode se adaptar para lidar com qualquer tarefa relacionada ao idioma.

### Casos de uso comuns:


#### Classificação

Verbos de instrução que são úteis para tarefas de classificação em nlp

  - «Classificar» o seguinte como uma propriedade física ou química
  - "Identificar" a ideia principal da passagem dada
  - "Categorize" a seguinte passagem como um mito ou um fato
  - 'Atribuir cada item' à sua categoria correspondente com base em suas características
  - «Listar» as cinco categorias principais do conjunto de dados fornecido.
  - «Agrupar» os artigos com base nas suas características comuns
  - 'Classificar' os dados com base na data em que foram coletados
  - 'Rotular' cada ponto de dados com sua categoria correspondente
  - "Dividir" os dados em subgrupos menores com base em suas semelhanças
  - 'Agrupar' os itens com base em sua semelhança nos recursos
  - 'Rotular' cada grupo com um nome descritivo.

#### Geração

Verbos de instrução que são úteis para tarefas de geração

  - 'Gerar' um novo texto com base nos parâmetros fornecidos
  - "Criar" um título curto e cativante com base no seguinte, com o objetivo de promover o produto
  - "Escreva" uma história usando as seguintes palavras-chave
  - "Formular" uma nova sentença com base na estrutura dada
  - 'Produzir' um novo texto que é semelhante em estilo à amostra dada
  - 'Construir' um novo texto que combina elementos de várias fontes
  - 'Compose' um novo poema baseado no tema dado
  - 'Construa' um novo texto combinando diferentes elementos de forma criativa
  - "Elaborar" uma nova narrativa baseada nas informações dadas
  - "Invente" uma nova história usando os seguintes elementos.

  - 'Criar um artigo de relatório' + 'com base na seguinte passagem'

## Estilo e Formato
---
Ao criar prompts, adotar um estilo específico é essencial por vários motivos. Um estilo bem definido ajuda das seguintes maneiras:

1. Consistência: Um estilo consistente torna seus prompts mais previsíveis e mais fáceis de entender para o modelo de IA. Ele garante que o modelo receba instruções claras e produza a saída desejada em vários prompts.

2. Clareza: Um estilo bem definido garante que suas instruções sejam claras e inequívocas. Isso reduz as chances de o modelo de IA entender mal seu prompt ou gerar respostas irrelevantes ou indesejadas.

3. Eficiência: Um bom estilo permite que você comunique seus requisitos de forma eficaz, reduzindo a necessidade de várias iterações ou ajustes no prompt. Isso economiza tempo e recursos.

4. Personalização: Ao adotar um estilo específico, você pode adaptar a saída para corresponder ao tom, formato e nível de detalhe desejados. Isso é particularmente útil ao tentar gerar conteúdo que adere a uma determinada voz de marca, padrão do setor ou público-alvo.

Ao escolher um estilo para seus prompts, considere os seguintes fatores:

1. Detalhe e especificidade: Seja o mais específico e detalhado possível sobre o contexto, resultado, comprimento, formato e estilo desejados.

2. Colocação da instrução: Coloque as instruções no início do prompt e use separadores como ### ou """ para separar a instrução e o contexto.

3. Exemplos: Forneça exemplos do formato de saída desejado, o que ajuda o modelo a entender melhor suas expectativas.

4. Evite a penumbra: Use uma linguagem concisa e precisa para minimizar a ambiguidade e a confusão.

5. Instruções positivas: Em vez de apenas dizer o que não fazer, diga o que fazer em vez disso.

Ao considerar cuidadosamente o estilo e elaborar seus prompts de acordo, você pode melhorar a qualidade e a relevância das respostas do modelo de IA, tornando suas interações com a IA mais eficientes e eficazes.
